# $20/Day System

## The Strategy: Freelance AI Services

**Target:** $20-50/day within 7 days, $200/day within 30 days.

---

## Step 1: Sign Up (Free)

1. **Fiverr.com** - Create account, add this landing page as your "portfolio"
2. **Upwork.com** - Apply to 10 jobs/day
3. **Toptal.com** - Apply (higher pay, harder to get in)
4. **Reddit r/forhire** - Post your services

## Step 2: Services to Offer (Ranked by Easy $)

| # | Service | Time to First $ | Avg Rate |
|---|---------|-----------------|----------|
| 1 | Data Entry / Excel | 1-3 days | $15-30/hr |
| 2 | AI Content Writing | 3-7 days | $25-50/hr |
| 3 | Virtual Assistant | 3-7 days | $20-40/hr |
| 4 | Simple Web Scraping | 5-14 days | $30-60/hr |
| 5 | Automation Scripts | 7-21 days | $40-100/hr |

## Step 3: Daily Action Plan

### Day 1-3: Set Up
- [ ] Create Fiverr account
- [ ] Create Upwork account  
- [ ] Set up profile with "AI Automation Specialist"
- [ ] Post 2-3 gigs on Fiverr ($30, $50, $100 tiers)

### Day 4-7: Get First Clients
- [ ] Apply to 10+ jobs/day on Upwork
- [ ] Send 5 cold emails/day to local businesses
- [ ] Respond to every inquiry within 1 hour
- [ ] Offer 20% discount for first 5 clients (get reviews)

### Day 8-30: Scale
- [ ] Raise rates 10% after 5 reviews
- [ ] Ask satisfied clients for referrals
- [ ] Add new service (e.g., "I built a bot for X")
- [ ] Create case study / portfolio page

## Quick Wins That Actually Work

### Fiverr Fast Start
1. Title: "I will [specific task] using AI"
2. Description: Clear, scannable bullets
3. Packages: $30 / $50 / $100
4. Deliver fast (24-48 hrs) → 5 stars → repeat clients

### Upwork Quick Tips
- Proposal cost: $0 (connects cost money)
- Custom proposal > template
- "I've done this exact task before" > "I'm a fast learner"

## The Math

| Daily Target | Hours @ $20/hr | Hours @ $30/hr |
|--------------|----------------|----------------|
| $20 | 1 hr | 45 min |
| $50 | 2.5 hrs | 1.7 hrs |
| $100 | 5 hrs | 3.3 hrs |

**Realistic:** Start at $20/hr, move to $40/hr after 10 jobs.
