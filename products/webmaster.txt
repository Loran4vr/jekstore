## ✅ PRE-LAUNCH CHECKLIST

### 1. DOMAIN SETUP
- [ ] Domain registered
- [ ] DNS configured
- [ ] SSL certificate active (HTTPS)
- [ ] Email forwarding set up

### 2. WEBSITE BASICS
- [ ] Homepage created
- [ ] About page written
- [ ] Contact page ready
- [ ] Privacy policy page
- [ ] Terms of service page

### 3. TECHNICAL
- [ ] Mobile responsive
- [ ] Page speed under 3 seconds
- [ ] All links working
- [ ] Images optimized
- [ ] HTML validated

### 4. SEO BASICS
- [ ] Title tags written
- [ ] Meta descriptions done
- [ ] Sitemap.xml created
- [ ] Robots.txt configured
- [ ] Google Search Console connected

### 5. CONTENT
- [ ] Blog posts ready (at least 3)
- [ ] Images have alt text
- [ ] Internal links working

### 6. LEGAL
- [ ] Privacy policy published
- [ ] Cookie consent banner
- [ ] Contact info accurate

## 🔍 SEO CHECKLIST

### On-Page SEO:
- [ ] Unique title for each page
- [ ] Meta description for each page
- [ ] H1 tag on every page
- [ ] Keywords in first 100 words
- [ ] Images with alt text
- [ ] Internal links

### Technical SEO:
- [ ] XML sitemap submitted
- [ ] Robots.txt configured
- [ ] Canonical URLs set
- [ ] 404 page custom
- [ ] Redirects working

### Local SEO (if applicable):
- [ ] Google Business Profile
- [ ] NAP consistent everywhere
- [ ] Reviews setup
- [ ] Location pages

---

## 📊 ANALYTICS SETUP

### Google Analytics:
1. Create account at analytics.google.com
2. Add tracking code to site
3. Set up goals:
   - Contact form submissions
   - Newsletter signups
   - Purchases

### Google Search Console:
1. Verify ownership
2. Submit sitemap
3. Check for errors
4. Monitor keywords
